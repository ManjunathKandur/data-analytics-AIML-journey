# 📘 Day 08 — Functions

Welcome to **Day 08** of my **100 Days of Data Analytics & AI/ML Challenge**.

Today, I explored one of the most fundamental concepts in Python programming: **Functions**. Functions enable us to write clean, organized, and reusable code by breaking large programs into smaller, manageable blocks.

Functions are extensively used in **Data Analytics, Machine Learning, and Artificial Intelligence** for data processing, automation, building reusable workflows, and creating scalable applications.

---

## 🎯 Learning Objectives

- Understand how to create and use functions in Python
- Pass data to functions using different types of arguments
- Return values from functions efficiently
- Write concise functions using lambda expressions
- Understand variable scope and accessibility
- Build reusable and modular programs
